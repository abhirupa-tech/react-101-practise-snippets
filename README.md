# React Practice Drills

100 small React exercises, built to keep fundamentals sharp in a world where everything is AI-assisted and vibe-coded.

The rules are simple: no AI autocomplete, no Copilot, time-boxed sessions. Just you and the problem. Each drill has a starter template (broken or empty) and a solution you can reveal once you've had a go.

**Feel free to fork this and make it your own.** Swap out my drills for whatever gaps you want to close.

## How to use

1. Clone (or fork, if you want your own copy) the repo:
   ```bash
   git clone https://github.com/<your-username>/react-101-practise-snippets.git
   cd react-101-practise-snippets
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Open the app in your browser and browse the list of drills to see the questions.
4. Solve each drill in your own editor's terminal (no AI autocomplete!) by editing the corresponding `*-default.tsx` file in `src/tasks/`, e.g. [001-the-liars-counter-default.tsx](src/tasks/001-the-liars-counter-default.tsx).

> **Note:** All drills are currently unsolved, so each `*-default.tsx` and its matching `*-solved.tsx` are identical for now. Solutions will be pushed progressively as they're completed — don't peek at `*-solved.tsx` until then.
